# smug
## roguelike in Java - tech demo

Configuration:

ctrl & f - change font

Keys:

arrows - move
ctrl & arrows - move
alt & arrows - move
numeric keypad - move
shift & arrows - scroll map
shift & numeric keypad - scroll map
o - open door
c - close door
shift & . - descend stairs
shift & , - ascend stairs
, - pickup
ctrl & q - quit
